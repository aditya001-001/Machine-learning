# 03. Data Visualization

This directory covers data visualization techniques using **Matplotlib** and **Seaborn**. 

Data visualization is key to Exploratory Data Analysis (EDA) because it helps us detect outliers, identify features' distributions, and observe linear or non-linear relationships.

---

## 📖 Contents

### [01_Matplotlib_Fundamentals.ipynb](file:///d:/ML/03_Data_Visualization/01_Matplotlib_Fundamentals.ipynb)
Covers basic plotting paradigms using Matplotlib:
*   **Line Plots:** Visualizing trends over time, sorting inputs, and grouping data for continuous plotting.
*   **Bar Plots:** Distribution of categorical fields.
*   **Pie Charts:** Proportional parts of a whole, showing percentage slices using `autopct`.

### [02_Seaborn_Advanced_Plots.ipynb](file:///d:/ML/03_Data_Visualization/02_Seaborn_Advanced_Plots.ipynb)
Focuses on advanced statistical plotting and syntax comparison:
*   **Scatter Plots:** Visualizing 2D numerical relationships.
*   **Box Plots:** Identifying outliers and looking at data distributions (quartiles).
*   **Correlation vs. Covariance:** Exploring strength of relationships between multiple feature columns.
*   **Matplotlib vs. Seaborn:** Syntax comparisons (e.g. using `hue` for multi-dimensional groupings).

### [03_Visualization_Exercises.ipynb](file:///d:/ML/03_Data_Visualization/03_Visualization_Exercises.ipynb)
A collection of practice problems and questions applying Pandas manipulations and visualization techniques on three distinct datasets (Superstore, Student Performance, and Google Play Store).

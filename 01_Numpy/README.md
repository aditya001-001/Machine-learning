# 01. NumPy Basics

This directory covers high-performance array operations and numerical computations using **NumPy** (Numerical Python).

NumPy is a critical foundation for machine learning as it enables fast vector and matrix calculations, which are standard for handling feature sets.

---

## 📖 Contents

### [01_NumPy_Basics.ipynb](file:///d:/ML/01_NumPy/01_NumPy_Basics.ipynb)
A comprehensive guide covering:
*   **Array Properties:** Creating homogeneous 1D & 2D arrays, list vs. array differences.
*   **Range Generation:** Generating numeric sequences with `arange()` and `linspace()`.
*   **Indexing & Slicing:** Multi-dimensional matrix indexing and row/column extraction.
*   **Random Number Generation:** Uniform/integer sampling (`randint`), random seeding, and unique element counts.
*   **Reshaping & Resizing:** Changing matrix dimension structures (`reshape` vs `resize`).
*   **Mathematical Foundations:** Manual calculation of Variance and Covariance using NumPy operations.

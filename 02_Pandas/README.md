# 02. Pandas Fundamentals

This directory covers data manipulation and analysis using **Pandas**. 

Pandas provides fast, flexible, and expressive data structures (DataFrames and Series) designed to make working with structured (tabular, multidimensional) data both easy and intuitive.

---

## 📖 Contents

### [01_Pandas_Fundamentals.ipynb](file:///d:/ML/02_Pandas/01_Pandas_Fundamentals.ipynb)
A consolidated, comprehensive guide combining base syntax, indexing models, and datetime formatting operations:
*   **Data Ingestion & Inspection:** Loading CSV files, checking metadata (`info()`, `dtypes`), and non-null values.
*   **Indexing & Slicing:** Selecting columns, slicing rows, and utilizing `.iloc` (integer-location) and `.loc` (label-location).
*   **Filtering:** Boolean masks to filter data rows based on criteria.
*   **Aggregations & Grouping:** Running custom aggregates with `agg()`, splitting data by categories with `groupby()`, and using `get_group()`.
*   **DateTime Operations:** Converting columns to datetime datatypes and extracting features like Year, Month, and Day.
*   **Cheat Sheet:** A reference guide summarizing core Pandas commands for quick lookup.
